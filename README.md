# Glassmorphism Calculator

A modern and responsive Calculator application built using HTML, CSS, and JavaScript. This project demonstrates core front-end development concepts including DOM Manipulation, Event Handling, and Input Validation while providing an attractive Glassmorphism user interface.

## Features

* Modern Glassmorphism Design
* Responsive Layout for Desktop and Mobile
* Basic Arithmetic Operations

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)
  * Percentage (%)
* Clear (C) Function
* Delete (⌫) Function
* Keyboard Support
* Animated Gradient Background
* Hover and Click Effects
* Floating Calculator Animation
* Real-Time Display Updates

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Concepts Implemented

### DOM Manipulation

* Dynamically updates calculator display.
* Retrieves and modifies user input.

### Event Handling

* Handles button click events.
* Supports keyboard input events.

### Input Validation

* Prevents invalid operator sequences.
* Prevents multiple decimal points in a number.
* Handles division by zero.
* Displays error messages for invalid expressions.
